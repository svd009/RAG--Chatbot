# RAG Q&A Chatbot 

A document-based Retrieval-Augmented Generation (RAG) chatbot using LangChain, FAISS, and OpenAI to answer context-aware questions from PDFs and websites.

## Features
- Document ingestion (PDF/web)
- Semantic search using FAISS
- LangChain-powered RAG pipeline
- Streamlit front-end
- RAGAS evaluation

## Stack
LangChain 路 FAISS 路 OpenAI API 路 Streamlit 路 RAGAS 路 Python

## Run
```bash
pip install -r requirements.txt
streamlit run app/app.py
